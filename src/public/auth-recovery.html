Ferramenta de Recuperação de Autenticação

Atenção: Esta ferramenta foi criada para resolver problemas de autenticação no frontend da plataforma Anunciar Grajaú.

Problema Detectado

Foi identificado um problema com tokens de autenticação em algumas sessões de usuários, resultando em erros 401 (Não Autorizado) ao acessar certas páginas do perfil.

Como Resolver

Para resolver este problema, adicione o script abaixo ao seu frontend:

1. Adicione o script ao seu HTML

<script src="https://api.anunciargrajaueregiao.com/auth-recovery.js"></script>

Coloque este script na seção <head> de todas as páginas que requerem autenticação.

2. Como funciona

O script faz o seguinte:

  • Verifica se há token problemático no localStorage
  • Faz uma chamada à API para obter um token de recuperação
  • Substitui automaticamente o token inválido
  • Monitora erros 401 e tenta recuperar a sessão automaticamente

3. Teste manual

Para testar manualmente a recuperação, você pode:

// No console do navegador
localStorage.setItem('token', '2tzoIYjxqtSE6LbFHL9mecf9JKM');
// Recarregue a página e verifique se o script detecta e corrige o problema
